Reports undeclared properties. Properties are undeclared if there not present in the properties section of the binding.

There are some exceptions:
  • Property name ends with: -controller
  • Property name starts with: #
  • Property name starts with: pinctrl-
And the following properties are allowed:
  • status
  • compatible
  • phandle
  • ranges
  • device_type
  • interrupts-extended
  • interrupt-parent
